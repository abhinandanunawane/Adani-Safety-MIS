How to use this dashboard

1. Navigation

  • Home — returns to the landing hero (hash #landing).
  • Dashboard Categories — choose a category to analyze related safety KPIs and performance (#categories → #cat=<id>).
  • Back to the grid — use Dashboard Categories in the header (or the browser Back control) to return to the domain list without losing your session.

2. Filters (toolbar)

Typical controls (availability depends on category):

  • Business unit — limit rows to one BU or keep All for the preview roster.
  • State — where the data model carries geography (e.g. Location Vulnerability).
  • Vertical / checkpoints — Field / O&M / Office / Projects roll-ups from the preview seed.
  • Calendar (from / to) — narrows the fact window; end month usually drives “latest” and Vs logic.
  • KPI list — multi-select checkboxes which KPIs drive tiles, trend line, radar, and matrix columns (hidden where the UX is map-only).
  • Reset — restores defaults for the open category (and clears saved KPI picks in local storage where applicable).

3. Versus (“Vs”) modes

  • The Vs control sets how current and base periods are chosen for trend lines, tiles, comparison bars, and table “Vs %”. Examples include last month, last quarter, prior year YTD, etc. The active window affects chart hints and grouped-bar comparison (Trend & Distribution View vs Comparison view).

4. Reading the charts

  • Each chart title includes a short hint (e.g. “12 mo”, “all BUs”). Hover or focus the i beside the title where shown to see filters and parameters for that visual.
  • Trend — monthly series for the selected KPI(s); SPI domains may show several lines.
  • By business / radar / bars — compare the same measure across BUs for the filtered window.
  • Location map — circle size reflects filtered row counts per BU (all contributing vulnerability KPIs combined).

5. Table view

  • Table view shows a BU × KPI matrix with Vs colouring. Use the small export controls on the table header for CSV (Excel-friendly) or JPEG (screenshot of the table block).

6. Downloading and exporting

  • Page (PDF / JPEG) — under the download (arrow) menu in the category toolbar: full page capture for stakeholder walkthroughs.
  • Filters report (PDF) — captures filter context and key blocks (when implemented in your build).
  • Chart JPEG — camera icon on each chart title exports that canvas/card as an image.
  • CSV — from the detail table exports (table toolbar icons).

Exports use your current filter and Vs settings. For governed numbers, reconcile against your Power BI dataset.
